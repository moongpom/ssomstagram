# ssomstagram
동덕여대 솜솜이 커뮤니티

## 서비스 소개
:동덕여대 커뮤니티 '동감'이 서비스 종료되며 아쉬운 마음에 커뮤니티를 만들게 되었습니다.
동덕여대 학우들을 칭하는 솜솜이와 인스타그램을 합쳐 '솜스타그램' 입니다. 

## 개발자
동덕여대 김지원/전혜인/정은영/정재운
